# CUDA – Paradigmas Avanzados de Programación (UAH)

Repositorio académico correspondiente a la asignatura **Paradigmas Avanzados de Programación**
de la Universidad de Alcalá.

El objetivo del repositorio es documentar de forma progresiva las prácticas realizadas
con **CUDA**, combinando explicación teórica, análisis de resultados y ejemplos de código
en C/C++ utilizando la **CUDA Runtime API**.
