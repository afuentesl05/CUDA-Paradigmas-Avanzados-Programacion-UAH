# CUDA-Paradigmas-Avanzados-Programacion-UAH
Repositorio académico para la asignatura Paradigmas Avanzados de Programación (UAH).  Incluye documentación detallada en formato README de las prácticas realizadas con CUDA,  análisis teórico de conceptos clave de computación paralela y ejemplos de código en C/C++  utilizando la CUDA Runtime API.
