# CUDA – Paradigmas Avanzados de Programación (UAH)

Repositorio académico correspondiente a la asignatura **Paradigmas Avanzados de Programación** de la Universidad de Alcalá.

El objetivo de este repositorio es documentar de forma progresiva las prácticas realizadas con **CUDA**, combinando explicación teórica, análisis de resultados y ejemplos de código en C/C++ utilizando la **CUDA Runtime API**.

## Estructura del repositorio

```
CUDA/
 ├── sesion_01/
 ├── sesion_02/
 ├── sesion_03/
 └── ...
```

Cada sesión contiene:
- Un `README.md` con la explicación teórica y resultados
- Una carpeta `src/` con el código fuente asociado

## Tecnologías utilizadas

- NVIDIA CUDA Toolkit 11.8
- C/C++
- Visual Studio 2019
- GPU NVIDIA GeForce RTX 3060
